Intended learning outcomes

  • Explain how the deformation gradient \(\boldsymbol{ F}\) represents the deformation of a volume element

  • Show that the small strain tensor, \(\boldsymbol{ \epsilon}\), is a reasonable strain measure for small rotations

Strain

The strain is a local measure of the shape-changes of a structure. As it is a local measure, its connection with the stress is not related to the structure's shape. Therefore, it is suitable to model the material behavior as the stress as a function of the strain: This material behavior can be characterized independently from the structure. For example, we can perform a laboratory test on a test sample to measure the material's stress-strain relationship. We can then apply this relationship to determine the local stress states when simulating an entire structure. But the question then is, how to define such a local strain measure? We have two key requirements

  1. It should describe how an infitesimal volume element (think cube) changes shape

  2. It should be independent of rigid body motion (translation and rotation)

In addition, for convenience, we would like to have a strain measure such that if it is zero there is no deformation

  1. It should be zero if there are no deformations.

The deformation gradient and the small strain tensor

The deformation gradient, \(\boldsymbol{ F}\), measures the gradient of the deformed coordinates, \(\underline{\boldsymbol{ x}}=\underline{\boldsymbol{ X}}+\underline{\boldsymbol{ u}}\) (the name displacement gradient is sometimes used for \(\boldsymbol{ F}-\boldsymbol{ I}=\ilpdiff{\underline{\boldsymbol{ u}}}{\underline{\boldsymbol{ X}}}\))

\[\begin{aligned} \boldsymbol{ F} = \underline{\boldsymbol{ x}}\otimes\underline{\boldsymbol{ \nabla}}_{\underline{\boldsymbol{ X}}} = \frac{\partial \underline{\boldsymbol{ x}}}{\partial \underline{\boldsymbol{ X}}} = \frac{\partial \underline{\boldsymbol{ u}}}{\partial \underline{\boldsymbol{ X}}} + \boldsymbol{ I} \end{aligned}\]

Here, \(\underline{\boldsymbol{ x}}\) are the deformed coordinates and \(\underline{\boldsymbol{ X}}\) the undeformed coordinates. That is, if we have the displacements \(\underline{\boldsymbol{ u}}\), then \(\underline{\boldsymbol{ x}}=\underline{\boldsymbol{ X}}+\underline{\boldsymbol{ u}}\). Based on this figure, it is clear that the deformation gradient describes how an infitesimal volume element changes shape. But it is not zero when the strain is zero - it is the identity tensor. Defining the strain as \(\boldsymbol{ F}-\boldsymbol{ I}\) solves this problem. However, consider a pure rotation of the square:

For the small rotation, \(\beta=5^\mathrm{o}\), we see that we obtain a change in the deformation gradient's off-diagonal (shear) terms. However, these are of equal magnitude but with opposite signs. So in the symmetric part of the deformation gradient is negligibly affected by the small rotation. So for small rotations, we can thus define a strain measure as

\[\begin{aligned} \boldsymbol{ \epsilon} = \left[ \underline{\boldsymbol{ x}}\otimes\underline{\boldsymbol{ \nabla}}_{\underline{\boldsymbol{ X}}}\right]^{ \mathrm{sym} } - \boldsymbol{ I} = \left[\frac{\partial \underline{\boldsymbol{ x}}}{\partial \underline{\boldsymbol{ X}}}\right]^{ \mathrm{sym} } - \boldsymbol{ I} = \left[\frac{\partial \underline{\boldsymbol{ u}}}{\partial \underline{\boldsymbol{ X}}}\right]^{ \mathrm{sym} } \end{aligned}\]

This strain measure is the small strain tensor, and we will use it throughout the linear continuum mechanics course.

Considering the larger rotation, \(\beta=30^\mathrm{o}\), even the diagonal (normal) terms are affected. Therefore, \(\boldsymbol{ \epsilon}\) is not suitable for large rotations.

In later courses dealing with nonlinear continuum mechanics, so-called finite strain measures can be introduced. Once example is the Green-Lagrange strain, \(\boldsymbol{ E}\), defined as

\[\begin{aligned} \boldsymbol{ E} = 0.5\left[\boldsymbol{ F}^{\mathrm{T}}\boldsymbol{ F} - \boldsymbol{ I}\right] \end{aligned}\]

A displacement field caused by a pure rotation is given by \(\underline{\boldsymbol{ u}} = \boldsymbol{ R}\underline{\boldsymbol{ X}}\), where \(\boldsymbol{ R}\) is a proper orthogonal rotation tensor. For this case, we obtain \(\boldsymbol{ F} = \boldsymbol{ R}\) and our two strain measures become

\[\begin{aligned} \boldsymbol{ \epsilon} &= 0.5 \left[\boldsymbol{ R} + \boldsymbol{ R}^{\mathrm{T}}\right] - \boldsymbol{ I} \neq \boldsymbol{ 0} \\ \boldsymbol{ E} &= 0.5\left[\boldsymbol{ R}^{\mathrm{T}}\boldsymbol{ R} - \boldsymbol{ I}\right] = \boldsymbol{ 0} \end{aligned}\]

So why do we not use this strain tensor instead? It involves square terms of the deformation gradient, and is therefore not linear. When solving structural problems, the linearity of the problem makes it much easier to solve. If we consider the linearization of the green-lagrange strain tensor, we would like to see how it is affected by a small deformation, i.e. \(\boldsymbol{ F} = \boldsymbol{ I} + \delta\boldsymbol{ F}\), where \(\boldsymbol{ \delta F}\ll\boldsymbol{ I}\) we have

\[\begin{aligned} \boldsymbol{ E} &= 0.5\left[\left[\boldsymbol{ I}^{\mathrm{T}}+\boldsymbol{ \delta F}^{\mathrm{T}}\right] \left[\boldsymbol{ I}+\boldsymbol{ \delta F}\right] - \boldsymbol{ I} \right] = 0.5\left[\boldsymbol{ I} + \boldsymbol{ \delta F} + \boldsymbol{ \delta F}^{\mathrm{T}} + \boldsymbol{ \delta F}^{\mathrm{T}}\boldsymbol{ \delta F} - \boldsymbol{ I}\right] \\ &\approx \boldsymbol{ \delta F}^{ \mathrm{sym} } = \left[\boldsymbol{ I} + \boldsymbol{ \delta F}\right]^{ \mathrm{sym} } - \boldsymbol{ I} = \boldsymbol{ \epsilon} \end{aligned}\]

where the fact that \(\boldsymbol{ \delta F}\ll \boldsymbol{ I}\) causes the term \(\boldsymbol{ \delta F}^{\mathrm{T}}\boldsymbol{ \delta F}\) to be negligible compared to \(\boldsymbol{ \delta F}\).
