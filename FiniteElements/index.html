Weak form

The weak form is an integrated form of the strong form, which makes the problem possible to approximate using the Finite Element Method. To obtain this form, we apply two "tricks":

  1. Scalar multiply the strong form with an arbitrary test displacement field v‾(x‾)\underline{\boldsymbol{ v}}(\underline{\boldsymbol{ x}})v​(x​)

  2. Integrate over the domain Ω\OmegaΩ

Because the test displacement field v‾(x‾)\underline{\boldsymbol{ v}}(\underline{\boldsymbol{ x}})v​(x​) is arbitrary, the following weak form of the boundary value problem is in practice equivalent to Equation (1). (There are some restrictions wrt. to the smoothness of the functions u‾(x‾)\underline{\boldsymbol{ u}}(\underline{\boldsymbol{ x}})u​(x​) and v‾(x‾)\underline{\boldsymbol{ v}}(\underline{\boldsymbol{ x}})v​(x​) from mathematics.) For brevity, from hereon we don't write out the dependence on x‾\underline{\boldsymbol{ x}}x​ and ϵ\boldsymbol{ \epsilon}ϵ for u‾\underline{\boldsymbol{ u}}u​, v‾\underline{\boldsymbol{ v}}v​, σ\boldsymbol{ \sigma}σ, u‾D\underline{\boldsymbol{ u}}_{ \mathrm{ D} }u​D​, and t‾N\underline{\boldsymbol{ t}}_{ \mathrm{ N} }t​N​.

Finite element form

The weak form in Equation (4) is also very difficult to solve, perhaps arguably more difficult than the strong form in Equation (1) since we need to deal with the arbitrary function v‾\underline{\boldsymbol{ v}}v​. The main issue for both of these forms, is that the solution is a continuous function u‾\underline{\boldsymbol{ u}}u​. It would be much easier to solve if we could somehow parameterize the function, e.g. if we could write that (in 1D) u(x)=a1x+a2x2+a3x3u(x) = a_1 x + a_2 x^2 + a_3 x^3u(x)=a1​x+a2​x2+a3​x3 and then just determine the coefficients aia_iai​. This is what we will do, namely that we will approximate both u‾\underline{\boldsymbol{ u}}u​ and v‾\underline{\boldsymbol{ v}}v​ using so-called shape functions:

u‾(x‾)≈N‾iu(x‾)aiv‾(x‾)≈N‾iv(x‾)δai\begin{aligned} \underline{\boldsymbol{ u}}(\underline{\boldsymbol{ x}}) &\approx \underline{\boldsymbol{ N}}^{ \mathrm{ u} }_i(\underline{\boldsymbol{ x}}) a_i \underline{\boldsymbol{ v}}(\underline{\boldsymbol{ x}}) &\approx \underline{\boldsymbol{ N}}^{ \mathrm{ v} }_i(\underline{\boldsymbol{ x}}) \delta a_i \end{aligned}u​(x​)​≈N​iu​(x​)ai​v​(x​)​≈N​iv​(x​)δai​​

where aia_iai​ and δai\delta a_iδai​ are the coefficients that we need to determine and N‾iu\underline{\boldsymbol{ N}}^{ \mathrm{ u} }_iN​iu​ and N‾iv\underline{\boldsymbol{ N}}^{ \mathrm{ v} }_iN​iv​ are shape functions that we postulate can be used to describe the solution and test space. So we restrict our solution to solutions that can be described by this discretization. Here, we will use the standard Galerkin method of having N‾i=N‾iu=N‾iv\underline{\boldsymbol{ N}}_i = \underline{\boldsymbol{ N}}^{ \mathrm{ u} }_i = \underline{\boldsymbol{ N}}^{ \mathrm{ v} }_iN​i​=N​iu​=N​iv​, which is described in detail in specific finite element courses.
